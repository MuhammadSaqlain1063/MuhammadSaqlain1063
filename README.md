# Digital Registrar Portfolio

This repository contains a uniquely themed personal portfolio website designed to look like an official academic document or a student ID card.

## 🚀 Overview

The "Digital Registrar" theme moves away from generic tech portfolios, leaning into the tactile and authoritative world of academic identification. It treats the digital space as an archival desk where professional credentials meet modern editorial flair.

## 📂 Project Structure

- **`index.html`**: The main portfolio page. It is a single-file implementation using Tailwind CSS.
- **`DESIGN.md`**: Detailed design strategy, including color palette, typography, and component logic.
- **`screen.png`**: A preview screenshot of the portfolio.

## 🛠️ Getting Started

To view the portfolio, simply open `index.html` in any modern web browser.

### Personalization

You can personalize this portfolio by editing `index.html` to update:
- Your Name
- Student ID / Registration Numbers
- Profile Portrait
- Project Archives
